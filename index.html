Marco Teórico

1. Continuidad Operativa: Pasar de una descripción conceptual hacia una justificación ingenieril más robusta del modelo propuesto. Profundizar el análisis técnico y crítico de los conceptos energéticos.

2. Rigor Científico: Fortalecer el uso de literatura científica especializada y journals indexados recientes, para incrementar el rigor académico y científico del proyecto en temas de resiliencia energética y confiabilidad operativa.

Este proyecto se basa en estudiar cómo asegurar que la EUNED siempre tenga energía para funcionar (resiliencia energética), pase lo que pase.

Buscamos soluciones en libros y estudios científicos actualizados para asegurar que el sistema sea confiable, no falle y la energía esté siempre disponible cuando se necesita.

Metodología

1. Definición de Variables: Definir con mayor precisión las variables críticas, KPI y criterios de aceptación para robustecer la validación cuantitativa de la propuesta.

2. Simulación: Desarrollar con mayor detalle la estrategia de simulación, explicando escenarios, herramientas, horizonte de análisis y lógica de evaluación.

3. Trazabilidad: Fortalecer la trazabilidad entre objetivos, actividades, herramientas y entregables para mejorar la coherencia y seguimiento técnico.

4. Modelo Sistémico: Integrar las herramientas diagnósticas, energéticas y operativas dentro de un modelo unificado para elevar el nivel de ingeniería aplicada.

Es el "paso a paso" de cómo vamos a resolver el problema:

  • Vamos a definir metas claras y medibles (indicadores).
  • Usaremos simulaciones por computadora para probar qué pasaría en la vida real.
  • Llevaremos un orden estricto desde el inicio hasta el final para asegurarnos de que la solución realmente funcione bien junta como un todo.
